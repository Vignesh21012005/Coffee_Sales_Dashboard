# 📊 Power BI Sales Dashboard

## 📌 Project Overview

This project demonstrates the creation of an interactive **Sales Dashboard** in Microsoft Power BI using a sales dataset containing 1,000 records. The dashboard helps analyze business performance through KPIs, trends, and interactive visualizations.

---

## 📂 Dataset Information

The dataset contains sales transactions with the following fields:

| Column | Description |
|---------|-------------|
| Order_ID | Unique order identifier |
| Order_Date | Date of the order |
| Region | Sales region |
| State | State of the customer |
| City | Customer city |
| Customer_ID | Unique customer ID |
| Customer_Name | Customer name |
| Product_ID | Product identifier |
| Category | Product category |
| Product_Name | Product name |
| Salesperson | Sales representative |
| Quantity | Number of items sold |
| Unit_Price | Price per unit |
| Discount_% | Discount applied |
| Sales | Total sales amount |
| Cost | Cost of goods sold |
| Profit | Profit earned |
| Payment_Mode | Payment method |
| Ship_Mode | Shipping method |
